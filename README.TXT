# Hotel-management-application(SPA)
This application is developed using angularjs, bootstrap, HTML5, CSS3, JavaScript, JSON, Node.JS, Express.Js, MySQL, Bower 

The Web site is all about hotel booking and manager tasks. The main purpose of the project is to create different pages for manager, employees and customers and keep them in sync. This application deals with manager activities like assigning task to employees, posting offers and managing events on home page, the manager can add and manage news employees and can check the entire customer booking history.
